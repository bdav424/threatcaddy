# ThreatCaddy Docs

## Reporting Procedure Entry Point

AI reporters producing analyst-style intelligence notes must start with
[`intel-note-reporting-procedure.md`](intel-note-reporting-procedure.md).

That procedure is the required source of truth for:

- reusing known-good Internal Word notes as locked visual templates
- preserving source-note markers and matching `See:` source notes
- preserving table colors, widths, captions, row padding, and spacing
- keeping noisy working tables out of customer-facing notes unless requested
- marking Word/render visual QA incomplete when it cannot be performed
